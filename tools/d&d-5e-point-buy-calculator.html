About This Calculator

The D&D 5e Point Buy Calculator helps you create balanced character builds using the official point-buy system. It allows you to:

  • Set custom point costs for different ability scores
  • Apply racial bonuses to specific abilities
  • Adjust minimum/maximum attribute limits
  • Track total points spent in real-time

How to Use

  1. Set your available points (standard is 27)
  2. Configure min/max attribute limits (typically 8-15)
  3. Input custom point costs for each ability score
  4. Select your race and assign racial bonuses
  5. Adjust ability scores and see real-time calculations

This tool ensures your character creation follows official D&D 5th Edition rules while allowing customization for homebrew campaigns.
